##LinkedList Operations
| Operation |  |  |
| ------ | ------ | ------ |
| Add | At the end | At beginning |
|   | O(N) | O(1) |
| Remove | At the end | At specific index or at the beginning |
|   | O(N) | O(1) |
| Search or Get | Get by index | Get by value (linear search) |
|   | Not available | O(N) |
